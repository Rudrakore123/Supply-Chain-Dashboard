# 📊 Supply Chain Performance Dashboard

## 📌 Overview
This Tableau dashboard analyzes supply chain performance using sales, profit, customer, and delivery data.

## 🛠️ Tools Used
- Tableau
- Excel

## 📈 Dashboard Features
- Monthly Sales Trend
- Sales by Category
- Sales by Market
- Top 10 Products
- Delivery Status Analysis
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Customers KPI

## 💡 Skills Demonstrated
- Data Visualization
- Dashboard Design
- KPI Reporting
- Business Analytics
- Data Storytelling
